## 🚀 Sobre o Projeto

Este projeto foi desenvolvido por **Wesly Felipe**, com o objetivo de criar um sistema completo de gestão empresarial (ERP), simulando rotinas reais de empresas.

O sistema roda via terminal e possui diversos módulos essenciais para o funcionamento de um negócio.

---

## 🧠 Funcionalidades

✔️ Cadastro de clientes  
✔️ Cadastro de fornecedores  
✔️ Gestão de funcionários (RH)  
✔️ Controle de estoque  
✔️ Controle financeiro (entradas e saídas)  
✔️ Registro de ponto  
✔️ Relatórios com data e hora  
✔️ Edição e exclusão de dados  

---

- Estrutura de dados (listas e dicionários)
